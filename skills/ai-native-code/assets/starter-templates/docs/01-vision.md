## 4. 如何使用本模板

1. **不要直接填**，让 Claude Code 走强制反问 ≥ 1 轮再起草
2. ≤ 5 条意图。第 6 条以后的想法先放 §3 反问历史段，等 Phase 3 再写成意图卡
3. 每条意图的"证据 / 证伪条件"必须可观测——能用日志 / 数据库 / 指标查到数字
4. 禁用主观词："用户体验糟糕"、"作者满意"、"界面好用"全部不合规
5. 定稿前用户必须审：每条意图你都同意吗？哪条最容易错？哪条不做项目就不成立？
