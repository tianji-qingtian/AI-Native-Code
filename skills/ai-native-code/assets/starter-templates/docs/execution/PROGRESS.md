## 更新规则

每次 Batch 状态变化时：

1. **开始**：把状态改 `in_progress`，填"开始"列为 `YYYY-MM-DD HH:MM`
2. **子任务级进度**：如 Batch 内部多个子任务被多个子代理分走，在该 Batch 下方补一小节子任务清单（打勾即可，不赘述）
3. **完成**：状态改 `completed`，填"完成"列，在"备注"加一句关键产出或踩坑
4. **中断**：状态保持 `in_progress`，在"备注"里写"已完成 XXX，待做 YYY"
