## 如何使用本模板

1. 编号严格递增：D-001, D-002, ...
2. **新决策追加到最上面**（最新的最容易看到）
3. 已否决的决策**不删**——保留历史，标 ❌
4. 大决策（架构 / 选型 / 不可逆）必须找用户拍板
5. 小决策（库选型、文件组织）Claude 可自主决定但必须留 ADR
6. 撞上规格冲突先入"待澄清"段，先按合理默认走，不阻塞
